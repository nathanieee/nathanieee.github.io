Programming In Java

Learn Java for those of you who want to learn the most popular Object Oriented Programming (PBO) concepts for developing applications.

Level : Intermediate

Java is a language created by James Gosling in the 1990s. Java emerged as a language that could be run on multiple platforms without the need for re-compilation. According to the TIOBE Programming Community Index, Java is still one of the most popular programming languages in the world. Oracle says 90% of the leading companies on the Fortune 500 list have used the Java language. In addition, Java can also be used to develop applications on desktop, web, mobile, embedded, and IoT platforms.

  • With the Java language, you can develop applications for desktop, web, mobile, to embedded and IoT platforms.
  • Whoever you are, if you want to become a Java Developer, you must have basic knowledge of Java programming such as Basic Java, Control Flow, Collection Type, and Object-Oriented Programming.
  • The use of the Java language is free (open source), so anyone can learn and use it.

Target audience and students

  • This class is intended for Intermediate student who want to learn the basics of Java programming language with reference to industry standards.
  • Classes can be attended by students who are IT literate so it is mandatory to have and be able to operate a computer well.
  • This class is designed for Intermediates so there are no prerequisites in prior understanding of programming. However, it would be better if you have basic knowledge of programming.
  • Students must be able to learn independently, be committed, really have curiosity, and be interested in the subject matter, because no matter how good this class material is, it will not be useful without students' seriousness to learn, practice, and try.
  • At the end of class, students can create Java programs using Intellij Idea IDEs or Online IDEs such as Glot.io or Replit.

Syllabus

  • Introduction to Java : An introduction to what Java is and why you should use Java, and you will get acquainted with the IDE for creating multiplatform applications. (55 minutes)
  • Basic Java : Explains the basics of the Java language, such as basic structures, data types, strings, operators, input output functions, and arrays. (2 hours 20 minutes)
  • Control Flow : Describes how control flow or flow works in the Java language, such as looping and branching. (55 minutes)
  • Collection Type : Describes various types of collection data such as lists, sets, and maps. (35 minutes)
  • Object-Oriented Programming (OOP) : Understand the concept of OOP (Object-Oriented Programming) to facilitate the development of a program such as classes and objects, methods, members, access modifiers, encapsulation, inheritance, and interfaces. (3 hours 55 minutes)
  • Common Class : Learn the classes that are often used in developing a program with the Java language such as generics, exceptions, input output, date time, and casting. (2 hours 45 minutes)
  • Case Study : Practicing the implementation of the Java language in a particular case study such as making a program to calculate water temperature, consonant vowels, and flat shapes. (2 hours)
