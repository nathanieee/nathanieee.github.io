Description

Microsoft Office

Learn how to get more work done, from anywhere on any device with Microsoft 365 and Windows 10. Discover how industry professionals leverage Microsoft 365.

Level : Beginner

Microsoft Office is a suite of applications designed to help with productivity and completing common tasks on a computer. You can create and edit documents containing text and images, work with data in spreadsheets and databases, and create presentations and posters. This course include the module for 3 Microsoft Office apps :

  • Microsoft Word
  • Microsoft Powerpoint
  • Microsoft Excel

Target audience and students

  • This class is intended for beginners who want to learn how to use Microsoft Office apps with reference to industry standards.
  • Classes can be attended by students who are IT literate so it is mandatory to have and be able to operate a computer well.
  • This class is designed for beginners so there are no prerequisites in prior understanding of programming. However, it would be better if you have basic knowledge of programming.
  • Students must be able to learn independently, be committed, really have curiosity, and be interested in the subject matter, because no matter how good this class material is, it will not be useful without students' seriousness to learn, practice, and try.
  • At the end of class, students can create Java programs using Intellij Idea IDEs or Online IDEs such as Glot.io or Replit.

Syllabus

  • Introduction to Microsoft : An introduction to what Microsoft is and why you should use Microsoft Office apps, and you will get acquainted with the interfaces for creating a professionals document (55 minutes)
  • Basic Office : Explains the basics of the Ms. Office interfaces, such as basic structures, and etc. (2 hours 20 minutes)
  • Expertise Office : Describes how to use Ms. Office like an expert. In-depth Explaining of the Ms.Office interface (55 minutes)

Study Tools

  • Students are required to have internet access through computers using Windows / Mac / Linux.
  • Minimum specifications of the student's computer:
    • Operating system: Windows, Linux, or MacOS (64-bit)
    • Processor: Intel Core i3 (Core i5 and above recommended)
    • RAM: 2GB (8GB Recommend)
    • Screen resolution : 1024 x 768 (Full HD 1920 x 1080 recommended)
  • Software:
    • Microsoft Office 365 : How to install will be taught in this class.
    • Diverse Idea Community : How to install will be taught in this class.
    • Any browser with internet connection.
